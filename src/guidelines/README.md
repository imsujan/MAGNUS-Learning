## üéØ Platform Design Principles

### **1. Learner-First Design**
Every feature prioritizes learner success over convenience or other factors.

### **2. 70-20-10 Learning Model**
- 70% hands-on practice and application
- 20% social learning and collaboration  
- 10% formal instruction

### **3. Portfolio-Driven Outcomes**
Every course produces job-ready portfolio artifacts.

### **4. Backward Course Design**
Start with outcomes ‚Üí Design assessments ‚Üí Build content

### **5. Universal Accessibility**
WCAG 2.1 AA compliance mandatory for all content.

### **6. Industry Alignment**
All skills taught must match real job requirements.

## üé® AMD Platform Brand Guidelines

### **Color Palette**

**Primary Colors:**
- Cyan Blue Accent: `#03A9F4` - Used for CTAs, highlights, links
- Dark Gray-Blue Header: `#455A64` - Navigation, headers
- White: `#FFFFFF` - Background, cards
- Light Gray: `#F5F5F5` - Secondary backgrounds

**Text Colors:**
- Primary Text: `#212121` (Dark Gray)
- Secondary Text: `#757575` (Medium Gray)
- Disabled Text: `#BDBDBD` (Light Gray)

**Semantic Colors:**
- Success: `#4CAF50` (Green)
- Warning: `#FF9800` (Orange)
- Error: `#F44336` (Red)
- Info: `#2196F3` (Blue)

### **Typography**

- **Headings:** Use semantic heading levels (H1, H2, H3)
- **Body Text:** 16px base font size, 1.5 line height
- **Reading Level:** 8th-10th grade for general audience content
- **Sentence Length:** 15-20 words average

### **Design Aesthetics**

- Clean, professional, modern
- Gradient backgrounds where appropriate
- Card-based layouts
- Generous white space
- Mobile-responsive design

## ‚úÖ Course Quality Standards

### **Minimum Requirements for Publication:**

- [ ] Learning outcomes clearly defined (3-5 measurable outcomes)
- [ ] 70%+ of time spent on hands-on practice
- [ ] All videos have accurate captions and transcripts
- [ ] Accessibility compliance (WCAG AA)
- [ ] At least 1 project per module
- [ ] Comprehensive capstone project
- [ ] Quality Checklist score: 80%+
- [ ] No critical failures (factual errors, missing captions, broken resources)

### **Excellence Standards (90%+):**

- Engaging instructor presence
- Real-world scenarios throughout
- Advanced accessibility features
- Exceptional project quality
- Outstanding learner outcomes
- Innovation in pedagogy
