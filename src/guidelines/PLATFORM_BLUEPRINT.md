## Platform Vision & Philosophy

**Mission Statement:**  
To create a transformative learning ecosystem where every learner acquires job-ready skills through hands-on practice, real-world projects, and industry-aligned instruction.

### Core Learning Principles

**1. Learner-First Design (LFD)**
- Every feature prioritizes learner success
- Clear learning outcomes displayed upfront
- Multiple learning pathways for different styles
- Immediate feedback and support systems

**2. Backward Course Design**
- Start with desired outcomes (job skills, portfolio artifacts)
- Design assessments that prove competency
- Build content that bridges the gap
- Ensure every lesson has measurable impact

**3. Universal Design for Learning (UDL)**
- Multiple means of representation (video, text, diagrams)
- Multiple means of engagement (interactive, hands-on, social)
- Multiple means of expression (projects, quizzes, discussions)
- Accessible to all learners regardless of background

**4. 70-20-10 Learning Model**
- 70% hands-on practice and application
- 20% social learning and collaboration
- 10% formal instruction and theory
- Platform enforces this ratio through content requirements

**5. Modular & Stackable Skill Blocks**
- Courses broken into micro-skills
- Skills can be combined into learning paths
- Learners build competency progressively
- Clear skill prerequisites and dependencies

**6. Real-World Simulation Projects**
- Every course includes industry-relevant projects
- Capstone projects mirror actual job scenarios
- Portfolio artifacts that impress employers
- Peer review mimics workplace collaboration

**7. Consistent Course Architecture**
- Learn â†’ Try â†’ Apply â†’ Assess â†’ Reflect
- Predictable structure reduces cognitive load
- Learners know what to expect in every course
- Quality consistency across all content

## Technical Architecture Overview

### Current Stack
- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Supabase (Auth, Database, Storage, Edge Functions)
- **API:** 13 RESTful endpoints (Hono server)
- **Hosting:** Supabase Functions
- **File Storage:** Supabase Storage (private buckets)

### Database Schema (KV Store)
- **Users:** Authentication and profiles
- **Courses:** Course metadata and structure
- **Chapters:** Individual lessons/videos
- **Learning Paths:** Curated course sequences
- **Enrollments:** User-course relationships
- **Progress:** Completion tracking
- **Certificates:** Achievement records

### API Endpoints
1. `POST /signup` - User registration
2. `GET /courses` - Fetch all courses
3. `POST /courses` - Create new course
4. `PUT /courses/:id` - Update course
5. `DELETE /courses/:id` - Delete course
6. `POST /enroll` - Enroll in course
7. `GET /enrollments` - User's enrolled courses
8. `POST /progress` - Update progress
9. `GET /progress/:courseId` - Get course progress
10. `GET /learning-paths` - Fetch all paths
11. `POST /learning-paths` - Create learning path
12. `POST /bulk-upload-course` - Folder-based course creation
13. `POST /certificates/generate` - Issue certificate

### Future Technical Enhancements
- **Search:** Elasticsearch/Algolia integration
- **CDN:** Cloudflare for video delivery
- **Analytics:** Mixpanel/Amplitude
- **Email:** SendGrid for notifications
- **AI:** OpenAI API integration for content generation
- **Real-time:** WebSocket for live sessions

## Learning Experience Flow

### **Learner Journey Map**

```
Discovery â†’ Enrollment â†’ Learning â†’ Assessment â†’ Completion â†’ Mastery
```

#### **1. Discovery Phase**
- Browse course catalog (filtered by topic, skill level, duration)
- View course details (outcomes, syllabus, instructor)
- Read reviews and ratings
- Preview sample lessons
- Check skill prerequisites
- View learning path recommendations

#### **2. Enrollment Phase**
- Click "Enroll Now"
- Access course immediately (no payment in current version)
- Receive welcome message from instructor
- Course added to "My Learning"
- Progress tracking initiated

#### **3. Learning Phase**
- **Module Structure:** Learn â†’ Try â†’ Apply â†’ Assess â†’ Reflect
  - **Learn:** Watch video, read notes
  - **Try:** Interactive practice exercise
  - **Apply:** Hands-on project task
  - **Assess:** Quiz or assignment
  - **Reflect:** Self-assessment questions
  
- Navigation tools:
  - Previous/Next lesson buttons
  - Progress bar
  - Module sidebar
  - Bookmarking
  - Note-taking (future)

#### **4. Assessment Phase**
- Complete quizzes (immediate feedback)
- Submit assignments (rubric-based grading)
- Participate in peer review (if required)
- Receive instructor feedback
- Revise and resubmit (if needed)

#### **5. Completion Phase**
- All modules marked complete
- Final capstone project submitted and approved
- Course completion badge earned
- Certificate generated (PDF download)
- Post-course survey (feedback)
- Next course recommendations

#### **6. Mastery Phase**
- Add certificate to LinkedIn
- Showcase capstone project in portfolio
- Apply skills on the job
- Mentor other learners (future)
- Become instructor (future pathway)
